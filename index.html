Some things about NO DQ, a different wrestling website.

  1. The website is extremely text heavy - the homepage content is almost entirely links
  2. Only using a very small amount of space on the homepage, looks more like a tumblr layout than website
  3. Navigation links are very clear and accessible
  4. Ads seem much more apparent because the content is crammed into a small space

By viewing source, I see mostly HTML that I recognize, as the site is heavily stylized with CSS. One HTML element that I recognize yet is the rel attribute, which I know is used to specify the relationship between the current document and a linked document. You can use this attribute to link to a CSS Stylesheet, for instance. Or to link to an author of the content.
